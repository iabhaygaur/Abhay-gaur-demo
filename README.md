# Python Data Science & Machine Learning Projects 🚀

Welcome to my repository showcasing end-to-end exploratory data analysis (EDA) and machine learning classification models built using Python.

---

## 📂 Projects Overview

### 1. Iris Dataset Classification 🌸
* **Objective:** Predict the specific species of an iris flower (Setosa, Versicolor, or Virginica) based on its sepal and petal measurements.
* **Key Steps:** * Performed Exploratory Data Analysis (EDA) with data visualization to understand feature distributions.
  * Built and evaluated multi-class classification models using Scikit-learn.

### 2. Titanic Dataset Survival Prediction 🚢
* **Objective:** Predict whether a passenger survived the Titanic disaster based on features like age, gender, passenger class, and cabin details.
* **Key Steps:**
  * Handled missing values and performed data cleaning.
  * Conducted feature engineering to prepare the data for training.
  * Implemented binary classification algorithms to evaluate model performance.
